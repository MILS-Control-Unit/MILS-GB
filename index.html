📤 Upload Grades by Item

Pick which mark-entry item you want to fill in from an Excel sheet. Only that item will be updated for the students already in this class list — everything else on their record stays untouched.

The file needs a ID or Student Name column to identify each student, plus a column with the item's own name (e.g. "Month 1 Q. 1") or a column titled Score. The downloadable template already has the right columns and current class list, so it's the easiest way to start. Rows that don't match any student in the current list are skipped and listed after the upload.

📊 Bulk Import Grades by Subject(s)

Import grades for one subject, multiple subjects, or all subjects at once from an Excel file.

You can select 2 or more subjects. Your Excel file should have columns for Student ID/Name and each selected subject.

📌 Import grades for 0 subjects in one Excel file.

Your file should have columns: Student ID (or Student Name), then one column per subject with the subject name as the header.

File format: First column should be Student ID or Student Name. For Single mode: one score column. For Multiple/All modes: each column header should match a subject name exactly. Only matching students will be updated; non-matching rows are reported after upload.
